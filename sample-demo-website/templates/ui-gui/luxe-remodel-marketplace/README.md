# Luxe Remodel Marketplace UI Template

Premium home-design marketplace GUI variant. This keeps the same high-level
split-search structure as the active skin, but shifts the tone toward upscale
remodeling, interior design, architecture, and concierge project intake.

## Visual Direction

- Light warm background
- Editorial service finder
- Dark bronze professional panel
- Softer service chips
- Premium remodel/design language
- Large hero image slot for kitchen, bath, studio, or showroom photography

## Best Fits

- Interior design studios
- Remodelers
- Kitchen and bath firms
- Architecture consultants
- High-touch home service brands
