# Pro Network Blueprint UI Template

High-contrast marketplace GUI variant for contractor networks, field services,
property maintenance, and service dispatch businesses. It uses the same split
hero pattern but feels more operational and work-focused.

## Visual Direction

- White marketplace header
- Cool gray workspace background
- Deep navy pro panel
- Compact service request form
- Strong service category chips
- More direct operations language

## Best Fits

- Contractor networks
- Property maintenance
- Field service teams
- Repair and installation companies
- B2B service marketplaces
