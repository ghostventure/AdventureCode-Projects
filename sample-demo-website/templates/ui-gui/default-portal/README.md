# Default Portal UI Template

Baseline UI/GUI skin for the sample leased-client portal. This is intentionally
generic and provider-free so it can be copied for a new client and customized
without changing the platform foundation.

## Intended Use

- Public landing page
- Client portal
- Manager workspace
- Admin/security views
- Platform setup and health views

## Swap Notes

- Keep the route structure intact unless a client explicitly needs fewer pages.
- Replace brand name, logo, colors, service labels, and imagery first.
- Keep billing, email, SMS, analytics, and webhook controls visually present but
  disabled until selected for that client.
- Keep the security and session controls visible in admin/security contexts.
