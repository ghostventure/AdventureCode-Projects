# Preview Notes

## First Pass Goals

- Keep the interface quiet, useful, and easy to scan.
- Use the same underlying routes and components.
- Swap colors, type direction, labels, and imagery without touching provider
  integrations.
- Keep dashboard and manager views denser than a marketing page.

## Required Checks

- Desktop screenshot
- Mobile screenshot
- Keyboard navigation spot check
- Color contrast check
- `npm run lint`
- `npm run build`
- `npm run smoke`
