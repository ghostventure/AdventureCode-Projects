Accessibility

Accessibility Statement

CLTCH.NTWRK is working to keep the platform usable for a broad range of users, devices, and assistive technologies. Accessibility is treated as an ongoing operational responsibility rather than a one-time checkbox.

Current Focus Areas

  • Readable color contrast and consistent typography
  • Keyboard-reachable form controls and navigation paths
  • Responsive layouts across mobile and desktop screens
  • Clear labels for primary booking, auth, and profile workflows

Known Limitations

Some pages are still being refined as the product evolves. Areas with heavy real-time state changes, inline status updates, or legacy layout code may need additional accessibility work over time.

Feedback

If you encounter an accessibility barrier, use the site contact channel and include the page, device, browser, and issue details. That feedback should be treated as a priority defect report.
