# CLTCH.NTWRK Contemporary GUI Redesign (2026-04-12)

## Request

Redesign the CLTCH.NTWRK GUI with a contemporary look for both Day and Night themes.

## Result

Completed.

The redesign was applied through the shared `cltch-boilerplate.css` layer so the visual system reaches the full static page set without rewriting every page individually.

Follow-up pass completed: the GUI was reshaped from a mostly themed static-card site into a stronger `Booking OS` visual architecture that better matches the feature-heavy runtime.

## Files changed

- `cltch-boilerplate.css`
  - Added a contemporary shared visual-system override.
  - Updated font import to include `Instrument Sans` and `Newsreader`.
- `mobile-web/cltch-boilerplate.css`
  - Regenerated by `npm run build:mobile` from the root shared CSS.
- `docs/DOCS_INDEX.md`
  - Updated to include this rollout note.
- `docs/PROJECT_STATUS.md`
  - Updated with the redesign and validation status.

## Visual direction

The new CLTCH.NTWRK GUI uses a cleaner booking-OS look:

- editorial typography for major page headlines
- neutral venue/booking color palette instead of heavy sci-fi glass
- refined Day theme with cream, white, denim, and rust-gold accents
- refined Night theme with deep neutral surfaces, warm gold, and clear blue accents
- stronger header structure
- cleaner page-width constraints
- unified card, panel, form, badge, chip, modal, and overlay styling
- more consistent button shapes and interaction states
- better desktop wrapping while preserving mobile behavior

## Booking OS architecture pass

The follow-up visual architecture layer added:

- pill-shaped sticky app header inside the page width
- left-edge status rail that gives the app a stronger product frame
- split hero layout with a high-contrast lead panel and lighter data panel
- larger editorial hero headlines and clearer room hierarchy
- feature hub reshaped into a command-room block
- dashboard stats restyled as compact operational instruments
- host, performer, booking, profile, and generated component cards given left accent rails
- workspace tabs normalized into segmented controls
- floating quick dock rebuilt as a centered command bar
- stronger card radius, spacing, and panel rhythm across desktop and mobile

## Day theme

The Day theme now uses:

- light cream page background
- white cards and panels
- muted denim/rust accents
- subtle grid texture in the page background
- stronger contrast for buttons, cards, and text

## Night theme

The Night theme now uses:

- deep neutral background
- dark slate cards and panels
- warm gold primary actions
- blue secondary/action accents
- reduced glare compared with the earlier dark/glass layer
