# iOS Expo Build Readiness - 2026-06-13

FoxHub mobile is prepared as an Expo/React Native app under `mobile/`.

## Local readiness

Run from the repo root:

```bash
npm run mobile:ready:ios
```

This command:

- regenerates mobile content and themes from the website source
- checks Expo dependency compatibility
- prints the public Expo config
- runs the mobile auth/profile smoke test
- exports the Expo web bundle

## EAS build path

Run from `mobile/`:

```bash
eas build:configure
eas build --platform ios --profile preview
```

Use `preview` first for an internal build. Use `production` later for TestFlight/App Store.

## Remaining external requirements

- Apple Developer Program account for device distribution/TestFlight/App Store
- Apple signing credentials handled by EAS during build setup
- Real EAS project link/project id generated by Expo during `eas build:configure`

## Current app identity

- Expo name: `FoxHub`
- Expo slug: `foxhub-mobile`
- iOS bundle identifier: `com.foxhub.app`
- URL scheme: `foxhub`
- Encryption export flag: `ITSAppUsesNonExemptEncryption=false`
